# Sale-Forecasting-model
Class project

[Kaggle](https://www.kaggle.com/code/huytriuquc/task-2-sale-forecasting-model-group-8)

Task 2: Sale Forecasting

-          Dữ liệu mua hàng (transaction) (event “purchased”)

-          Event data: “view_item”, “add-to-cart” (ATC)  à new dataset

-          Train data: 2025

-          Test data: 1/2026 (blind) chỉ tính event “purchased”

-          Output giao nộp: Bảng dữ liệu bao gồm 3 cột “location”, “item_id” và “prediction”

-          Tính metric đánh giá trên những cửa hàng (location) có phát sinh giao dịch. Không tính những mặt hàng có sale_status = 0

-          Metrics:

o   MAE trên doanh số

o   MAE trên doanh thu

o   MAPE trên doanh số (chính)

o   MAPE trên doanh thu
